## [0.6.0] - 2026-04-11

### Added

**Unified Kanban** — A beautiful, native VS Code experience for your kanban boards.

Unified Kanban renders your `.md` kanban boards beautifully in VS Code. Built for Obsidian kanban syntax compatibility with a beautiful, native VS Code experience.

- Seamless kanban visualization — Open any markdown file with kanban syntax and view it as an interactive board with drag-and-drop columns and cards
- Obsidian-compatible — Full support for Obsidian kanban syntax, including tags, dates, and metadata
- Native VS Code integration — Built with the VS Code extension API for a polished, performant experience that feels right at home in your editor
- Customizable settings — Configure default column widths, date formats, accent colors, card insertion methods, and more
- Multiple view modes — Switch between Board, Table, and List views depending on your workflow
- Keyboard shortcuts — Quick access to create new boards, archive cards, and toggle views
- Archive management — Easily archive completed cards with optional date stamps
